環境設定

設定項目 解説／設定値
Redmine のパス Redmine の相対URL。ホスト名を含まない「/」から後のパスを設定します。
Redmine がルートを使用している場合は「/」のみとなります。
リバースプロキシを設定している場合はプロキシのパスを設定します。
例: /redmine/
Redmine絶対パス
(オプション)		 Redmineの絶対URL。プロトコル、ホスト名を含むURLを設定します。
リバースプロキシを使用している場合のみ必要です。

例: http://www.redmine.org/
APIアクセスキー
(オプション)		 Redmineの個人設定で取得できるAPIアクセスキーを設定します。
設定しない場合は未ログイン状態となります。

例: d557be383b862e3f852be39ae1716a1f
カスタムフィルタ
(オプション)		 Redmine REST API にしたがった任意のクエリを設定します。
通常はRedmineでカスタムクエリを作成しそのIDを query_id パラメータで指定します。
ボタンが画面上部に追加されます。

アイコン タイトル クエリ





例：山田さん担当チケット 例：query_id=20
